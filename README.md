
# Github Users 

Search Github Users. Get details displayed in charts.

## Tech Stack

React, React Router Dom, React Icons, Styled Components, Fusion Charts, Auth0
  
## Deployment

Netlify

  
## Environment Variables

To run this project, you will need to add the domain and client ID from Auth0 to your .env file

`REACT_APP_DOMAIN`

`REACT_APP_CLIENT_ID`

  
## Run Locally

Clone the project

```bash
  git clone https://github.com/rajatetc/github-users.git
```

Go to the project directory

```bash
  cd github-users
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

  
## API Reference

https://api.github.com/

For unauthenticated requests, the rate limit allows for up to 60 requests per hour. Unauthenticated requests are associated with the originating IP address, and not the user making requests.
